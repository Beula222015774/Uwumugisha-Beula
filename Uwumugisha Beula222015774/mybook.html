        Attributes of User Table
        Here are the descriptions or purposes of the attributes/columns used in User table:

        userid: The column which are represents id of user which is unique value called PRIMARY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of Product table.

        username: The column which are represents the name of the user like John.


        Email: The column which are represents the email of the user.

        Gender: The columns which are represents the gender of the user may be female or male .

        Attributes of Course Table
        Here are the descriptions or purposes of the attributes/columns used in Course table:

        courseid: The column which are represents id of the course which is unique value called PRIMARY KEY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of the course table.


        userid: The column which are represents id of the user which is references to the user table value called FOREIGN KEY. It is a column of user table which has relationship between course table.


        coursename: The column which are represents the name of the course like web development.


        coursecode: The column which are represents the code of the course.
